## Setup Instructions

### Prerequisites
- Node.js (>= 16.x)
- npm (Node Package Manager)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/prudentbird-dev/hng-stage-zero.git
   cd hng-stage-zero
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

4. Start the server:
   ```bash
   npm start
   ```

5. Access the API at:
   ```
   http://localhost:3000
   ```
